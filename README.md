# E2e-boilerplate
    
e2e-boilerplate.
    
## Getting Started
    
1. git clone git@github.com:e2e-boilerplate/e2e-boilerplate.git
    
2. cd e2e-boilerplate
    
3. npm install
    
4. npm run build
    
5. npm run test
